* Burrows-Wheeler Transformation
* Ferragina-Manzini Index

### HW3: Global and Local Alignments of Sequences (C++)
* Needleman-Wunsch
* Smith-Waterman

### HW4: Multiple Sequence Alignment (C++)
* Sequence to Profile Alignment
